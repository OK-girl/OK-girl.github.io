左边div固定宽度，右边div自适应撑满剩余宽度布局

大胖 发布于 2019-12-11

页面的排版布局上，为处理不同浏览器、不同显示器的网页结构展示，自适应的布局很必须。左边固定宽度，右边自适应剩余宽度常常作为很简单的页面布局结构方式。

  • HTML

      <div class="container">
      <div class="c-left"></div>
      <div class="c-right"></div>
  </div>

  • css

    • 1.使用float(BFC)
       .container>.c-left{
     width:100px;
     float:left;
 }
 /*不设置具体宽度的时候，可是配合使用margin-left和overflow："hidden"也可以实现右边自适应*/
 .container>.c-left{
      margin-right: 20px;
      float:left;
  }
 .container>.c-right{
     overflow:hidden;
 }
    • 2.使用div模拟table(IE+8以上)
        .container{
    display: table;;
    width: 100%;
  }
  .container>div{
      display: table-cell;
  }
  .container>.c-left{
      width: 100px;
  }
    • 3.使用flex
        .container{
    display: flex;;
  }
  .container>.c-left{
      width: 100px;
      flex: none;
  }
  .container>.c-right{
      flex: 1;/*这里设置为占比1，填充满剩余空间*/ 
  }
